What i needed for this was a whole load of nothing. i paid a total of like 16 bucks for the domain name and the setup of that was a bit of a pain but thats where i started for some reason.

i was wrong

i set the page up, it looked great, and then the second i deployed the ready version i was told that i cant continue to display the page because i ran out of credits. so much for free website hosting. awful. so luckily i pushed the page to be a github repo for that setup which was easy enough, then i couldnt use those files and turns out its easier to just delete it. so the first revision is fully gone now. then i had this page recreated in an easier way and hosted it directly on github, they wont force me to pay to make a change. right... well we will find out if you cant access this. or i mess up the page somehow. anyways, i restarted basically. had to delete the project off netlify and redirect my domain name to the github page and it was simple as that. i did this all mid way through an overwatch game and as soon as i came into work. now im gonna fine tune it.
